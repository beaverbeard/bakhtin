# Бахтин — golden-харнесс тестирования

Методика повторяемой проверки качества выхода. Четыре слоя: от дешёвого
автоматического до дорогого LLM-судьи. Генератор сам по себе не стерилен —
харнесс это второй рубеж обороны.

## Слой 1 — Структурный чекер (автомат, `check.py`)

Ловит механизируемые провалы без LLM:
- **Выдуманные числа** (цифры + русские числительные словами) не из allow-листа
  источника.
- **Хард-баны** AI-маркеров из базового профиля (regex).
- **Длина** вне диапазона формата.

```
python3 check.py --draft draft.txt --allow "две,третий" --min 1500 --max 2500
```

`--allow` — числа, которые ЕСТЬ в материале (их вводить можно). Всё остальное
числовое — флаг. Exit 1 при флагах. Покрывает: цифры, числительные словами,
собирательные (пятеро) и кратные (втрое), исключает идиомы-гиперболы («тысячу
раз»), нумерацию тредов (1/8), порядковые-маркеры.

## Слой 2 — Анти-AI детектор структурного слопа

Выход прогоняется через анти-AI вычитку (для русского — например
[beaverbeard/slopotron](https://github.com/beaverbeard/slopotron)). Слой 1 ловит
точечные лексические маркеры, слой 2 — структурные (монотонный ритм, балансные
«не X, а Y», AI-триады, водянистость). Voice-выходы дают заметно более высокую
плотность слопа — им этот слой нужнее.

## Слой 3 — LLM-судья по рубрике (субагент)

Прогон по родной рубрике скила. Промпт судьи (фиксированный):

> Оцени текст по рубрике 1-10 каждый: Хук, Голос, Угол, Панч, Верность исходнику,
> Фактическая точность. Флаг любой критерий < 6. Верни оценки + одно главное
> замечание.

## Слой 4 — Adversarial-скептик (субагент)

Дружелюбный судья переоценивает текст. Скептик «по умолчанию ищу изъян» ловит:
фабрикацию сверх чисел (выдуманные кейсы/имена/цитаты), логические дыры
(корреляция как причинность, ложные дихотомии), структурный слоп, слом регистра,
распад жанра по длине. Промпт:

> Найди проблемы, не похвали. По умолчанию считай, что текст содержит изъян.
> Перечисли конкретно: фабрикации, AI-маркеры/слоп, логические дыры, нарушения
> жанра.

## Критерий зелёного харнесса

- Слой 1: 0 флагов по числам не из материала, 0 хард-банов, длина в диапазоне.
- Слой 2: плотность слопа ниже порога.
- Слой 3: средняя рубрика ≥7, ни один критерий < 6.
- Слой 4: 0 фабрикаций-обманов (расширения мысли допустимы).
